# UNP_PROJECT_BI_WITH_POWER_BI_HR_ATTRITION_ANALYSIS

i did this project in collaboration with UNP , This Power BI file is a digital tool used to understand why employees leave a company and how to keep them. It works like a smart report that takes messy HR information and turns it into easy-to-read charts and graphs. Inside, it uses special math formulas to calculate things like how many people are quitting and the total number of staff. A key feature is a "Decomposition Tree," which lets you click through different departments or job titles to find the exact reason for high turnover. The report looks professional because it uses a clean business theme and includes helpful pictures about HR strategies and employee retention. Basically, it’s a master file that helps a team present data-driven ideas to bosses on how to make the workplace better and stop people from leaving.
